# 第 11 课　Agent 智能体：会自己用工具的 AI

|**🎯 本课目标**<br>1\. 理解 Agent 和普通应用最本质的区别；<br>2\. 看懂 Agent 的“思考\-行动”循环；<br>3\. 学会给 AI 配“工具”，做一个会联网搜索的智能体；<br>4\. 知道 Agent 和工作流分别该用在什么场景。|
|---|

**前面的应用，流程都是我们事先定死的。**Agent 是个质的飞跃：你只给目标，它自己决定分几步、每步用什么工具去完成。

## 11\.1　Agent 到底特别在哪

普通应用是“你规定好流程，它照做”；Agent（智能体）厉害在：你只给它一个目标，它自己决定分几步、每步用什么工具去完成。

举个例子：你问“北京现在天气怎么样，适合穿什么”。

- 普通 AI：它没有实时数据，只能瞎猜或说不知道；

- **Agent：**它会想“我需要查天气”→ 自动调用‘天气工具’拿到实时数据 → 再根据温度给出穿衣建议。整个过程它自己安排，不用你一步步教。

|**💡 小贴士**<br>一句话区分：工作流是“你画好地图，它照着走”；Agent 是“你说出目的地，它自己找路”。后者更灵活，但也更不可控。|
|---|

## 11\.3　工具：给 AI 装上“手和脚”

**工具** 就是让 AI 能调用的外部能力。没有工具，AI 只会“动嘴”；有了工具，它才能“动手”去查、去算、去办事。

|**工具类型**|**举例**|**能让 AI 做什么**|
|---|---|---|
|内置工具|联网搜索、网页读取、计算器、画图|查实时信息、做运算、生成图片|
|自定义工具|把你公司的接口（API）接进来|查订单、查库存、写入系统|
|工作流转工具|把第 10 课做的工作流当成一个工具|复用已经做好的复杂能力|

|**💡 小贴士**<br>新手先玩内置工具，尤其是“联网搜索”，能立竿见影地解决“AI 不知道最新消息”的问题。自定义工具属于进阶，需要一点接口知识，可以以后再学。|
|---|

## 11\.4　实操：做一个“会上网的研究助手”

1. 创建应用，类型选“Agent / 智能体”。

2. 在提示词里设定它的角色和工作方式，例如：

|**示例·Agent 提示词**<br>*你是一名严谨的资料研究员。当用户的问题涉及最新信息时，先用搜索工具查证，再综合给出带出处的回答；信息不确定时如实说明，不要编造。回答分点、简洁。*|
|---|

1. 在“工具”区，添加一个“联网搜索”类的内置工具（按提示授权或填好配置）。

2. 确认选好对话模型——Agent 对模型的“推理能力”要求较高，尽量选能力强一点的模型。

3. 在预览区问一个需要最新信息的问题，比如“最近有什么 AI 领域的新闻”。

4. 观察它的执行过程：你能看到它“调用了搜索工具 → 拿到结果 → 整理成回答”的全过程。

|**⚠️ 避坑提醒**<br>Agent 要多次思考和调用工具，速度比普通应用慢、消耗也更多。它适合“确实需要动手查/算”的问题；任务很简单时用聊天助手反而更快更省。别为了用 Agent 而用 Agent。|
|---|

## 11\.5　Agent 和工作流怎么选

||**工作流**|**Agent**|
|---|---|---|
|谁定流程|你（步骤固定）|AI（自己临场决定）|
|可控性|高，结果稳定|较低，更灵活但有不确定性|
|速度/成本|较快、较省|较慢、消耗更多|
|适合|流程明确的标准任务|开放、需随机应变的任务|

**选择口诀：**流程你都想清楚了 → 用工作流；连你自己都说不准要几步、要看情况临场决定 → 交给 Agent。

## 11\.6　用好 Agent 的几条注意事项

- 提示词写清楚“什么情况用什么工具”，减少它乱用工具；

- 尽量选推理能力强的模型，Agent 的表现高度依赖模型水平；

- 上线前多测试各种刁钻问题，观察它会不会绕弯、会不会用错工具；

- 对结果的准确性有硬要求的关键场景，优先用可控的工作流，而不是 Agent。

|**❓ 常见疑问**<br>问：Agent 会不会乱用工具、给错答案？<br>答：有可能。它毕竟是“自己拿主意”，可控性不如工作流。降低风险的办法：提示词写清楚（什么情况用什么工具）、选推理强的模型、上线前多测试。|
|---|

|**❓ 常见疑问**<br>问：Agent 能挂知识库吗？<br>答：能。可以把知识库（检索）也作为它的一种“工具”，让它在需要时自己决定去查资料。这样就既能联网、又能查内部资料，很强大，但也更要测好。|
|---|

|**✏️ 课堂练习**<br>1\. 做出“会上网的研究助手”，问它几个需要最新信息的问题，观察工具调用过程。<br>2\. 对比：同一个问题，分别用“普通聊天助手”和“这个 Agent”回答，看差别在哪。<br>3\. 思考你的工作里，哪些任务适合 Agent、哪些更适合工作流，各举一例并说明理由。|

---

**本课小结：**Agent 能自己决定步骤、调用工具，核心是“思考\-行动”循环；工具给 AI 装上手脚；它灵活但慢、不够可控；流程明确用工作流，需随机应变才用 Agent。
